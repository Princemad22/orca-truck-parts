# ORCA Truck Parts Website

Complete truck parts e-commerce website with Arabic/English support and dark mode functionality.

## Features
- Dual language support (Arabic/English)
- Dark/Light mode toggle
- Admin panel with role-based access
- Product catalog with categories
- News/blog section
- Contact form
- Testimonials section
- Partner/supplier showcase
- Responsive design

## Technologies Used
- PHP 8.x (Core PHP)
- MySQL database
- HTML5, CSS3, JavaScript
- Bootstrap 5
- jQuery
